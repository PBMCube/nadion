Class: StateMachine

Nadion. StateMachine

Class providing a state machine for the Nadion Phaser add-on. Takes an array of objects that define the states and events, like such: 
states = [
	{
		'name':'working',
		'initial':true,
		'events':
		{
			'bored':'coffee',
			'call_for_meeting':'meeting',
		}
	},
	{
		'name':'coffee',
		'events':
		{
			'break_over':'working',
			'call_for_meeting':'meeting'
		}
	},
	{
		'name':'meeting',
		'events':
		{
			'meetings_over':'working'
		}
	},
];
and a receiver ('this' object) for the callbacks. Callbacks are made on the receiver object, have the same name as the new state, and take one parameter: the event. e.g. 
function meeting( event ) { this.attendMeeting(); }

new StateMachine(states, receiver)

Parameters:
Name Type Description
states Array An array of objects that define the states that the state machine can be in, what events can occur and what states those events result in.
receiver Object The object that will receive calls on state changes. (Typically 'this' when a state machine is included as a property of an object).
